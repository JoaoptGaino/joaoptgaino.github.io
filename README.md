# Joao Pedro Theodoro Gaino Portfolio

Personal portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment Variables

Use `.env.example` as a reference. Keep real values only in your local `.env` file or in Vercel project settings.

```bash
NEXT_PUBLIC_SITE_URL=
NEXT_PUBLIC_GITHUB_USERNAME=
NEXT_PUBLIC_CONTACT_EMAIL=

CONTACT_TO_EMAIL=
CONTACT_FROM_EMAIL=
RESEND_API_KEY=
```
